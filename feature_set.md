# Lanternfish
## Functionalities
* LCD
* RTC + cr2032 for time keeping
* Manual-auto mode (manual mode controlled by knob)
* 0-100% switching
* Polarity protection
* Screen turns off after timeout

## User interface
* Settings
  - Date
  - Time
  - Presset (sun, inverted sun, user 1, user 2, user 3)
* Schedule
  - button to add a level

## Hardware
* Buttons
  - Left-right
  - Add
  - Manual/auto
  - knob
